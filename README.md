# Weather App
This is a simple project which aims to provide weather condition of a given city just by typing its name.

# Installation

1) Download the above files.
2) Move to this file by typing in terminal:
  ``` cd <name of file>/weather```
3) Make a new tab in terminal and type in the following:
  ```cd <name of file>/Server ```
4) Install the dependencies by (in both tabs):
   ``` npm i ```
5) Type the following (in the Server tab) to set up node at localhost:3000 
   ``` node server.js ```
 and the following in the other tab:
  ``` npm run dev ```
6) Now the server is set up locally. Open localhost:5173 in your browser and enjoy the service!
